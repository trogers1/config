# XDG Configs

This directory contains application configuration that belongs under `~/.config/`.

"XDG" refers to the XDG Base Directory convention, which standardizes where
user-specific config files live on Unix-like systems.

In this repo:

- `xdg/nvim` maps to `~/.config/nvim`
- `xdg/opencode` maps to `~/.config/opencode`

The setup script symlinks each directory in `xdg/` into `~/.config/`.
